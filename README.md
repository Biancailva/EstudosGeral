# EstudosGeral
Codigos usados para estudos, codigos das disciplinas praticas de Estrutura de Dados e Programação Orientada a Objetos do curso de análise e desenvolvimento de sistemas.

Temos assuntos de estrutura de dados e programação orientada a objetos que envolvem os seguintes assuntos:
-Vetores
-Pilhas
-Filas
-Fila encadeada
-Matrizes
-Matrizes Tridimencionais
